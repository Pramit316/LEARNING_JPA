This repository is a small learning project created to understand Java Persistence API (JPA) with Spring Boot and PostgreSQL.

The main purpose of this project is to practise core JPA concepts such as:

Creating entity classes
Mapping Java classes to database tables
Using primary keys with @Id
Generating IDs using @GeneratedValue
Understanding SEQUENCE, IDENTITY, and TABLE generation strategies
Connecting a Spring Boot application to PostgreSQL
Learning how Hibernate creates and manages database tables

This is not a production-level project. It is mainly built for hands-on practice while learning how JPA and Hibernate work behind the scenes.

Technologies Used
Java
Spring Boot
Spring Data JPA
Hibernate
PostgreSQL
Docker
Purpose

I created this repository as part of my learning journey to better understand how backend applications interact with relational databases using JPA.
